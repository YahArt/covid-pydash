Purpose

Covid PyDash was created as a proof of concept Web Application during my bachelor thesis related to Corona Data Visualizations. The main goal of Covid PyDash is to give users the possibility to create their personalized Corona Dashboards. Do note that this Application is entirely non profit and does not pursue any monetary intentions
