Welcome to Covid PyDash where you can create your own personalized Corona Dashboard!

As a starting point you have a few possibilities to choose from:

Create a dashboard

Create your own dashboard

You can create a Corona Dashboard either from a template or completely from scratch. While you are there you have the option to keep creating dashboards or open the dashboard directly once it is created. Once you have opened a dashboard you can dynamically add all kind of different widgets related to Covid. You also have the possibility to activate edit mode. Once you are in edit mode you can move, edit and delete your widgets. Of course you also have a saving functionality, so don't worry about loosing your progress

 Dashboard Overview

Manage your created dashboard

In the dashboard overview you can easily manage all your created dashboards. This includes functionality such as opening the dashboard, deleting and sharing functionality
